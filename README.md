# Website Institucional - Agência Nexus

Este repositório contém o código-fonte do website institucional desenvolvido para a **Nexus**, uma agência de marketing digital focada em impulsionar o crescimento de negócios através de estratégias inovadoras. O projeto foi criado para apresentar os serviços, diferenciais e cultura da agência de forma moderna e responsiva.

## 🚀 Tecnologias Utilizadas

Este projeto foi desenvolvido utilizando as seguintes tecnologias:

- **[React](https://react.dev/)**: Biblioteca JavaScript para construção de interfaces de usuário.
- **[Tailwind CSS](https://tailwindcss.com/)**: Framework CSS utilitário para estilização rápida e responsiva.
- **[Framer Motion](https://www.framer.com/motion/)**: Biblioteca para animações fluidas e gestos.
- **[Lucide React](https://lucide.dev/)**: Coleção de ícones SVG limpos e consistentes.
- **[Vite](https://vitejs.dev/)**: Ferramenta de build e servidor de desenvolvimento rápido.

## 🎯 Funcionalidades e Seções do Site

O site foi projetado para apresentar a agência Nexus, seus valores e serviços de forma clara e objetiva. As principais seções incluem:

-   **Sobre (About)**: Apresenta a filosofia da agência com o lema "Mais do que uma agência, o seu time de marketing", destacando sua essência no audiovisual e na estratégia.
-   **Serviços Especializados**: Detalha as quatro principais áreas de atuação da Nexus:
    *   Planejamento e Estratégia
    *   Criação e Audiovisual
    *   Design e Posicionamento
    *   Performance e Expansão
-   **Diferenciais (Benefícios)**: Exibe os benefícios de contratar a agência, como foco em estratégia, tráfego qualificado e parceria real com os clientes.
-   **Contato (Contact)**: Inclui uma chamada para ação (CTA) clara, incentivando potenciais clientes a solicitar propostas e a entrar em contato via WhatsApp.

## 🔧 Como Executar o Projeto

Siga os passos abaixo para rodar o projeto em sua máquina local:

1. **Clone o repositório:**

    > **Nota:** Substitua `seu-usuario/Nexus.git` pelo caminho real do seu repositório.

   ```bash
   git clone https://github.com/seu-usuario/Nexus.git
   cd Nexus
   ```

2. **Instale as dependências:**

   ```bash
   npm install
   # ou
   yarn install
   ```

3. **Inicie o servidor de desenvolvimento:**

   ```bash
   npm run dev
   ```

4. **Acesse o projeto:**
   Abra o navegador e acesse `http://localhost:5173` (ou a porta indicada no terminal).

## 📦 Build para Produção

Para gerar a versão otimizada para produção:

```bash
npm run build
```

Os arquivos gerados estarão na pasta `dist`.
